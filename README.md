# Chandra Associates

A React + Vite website for Chandra Associates.

## Local development

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app locally:
   ```bash
   npm run dev
   ```
3. Open the local URL shown by Vite (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
```

The production site is generated into `dist/`.

## Deploy your website

You can deploy the generated `dist/` folder to any static host:

- Netlify
- Vercel
- GitHub Pages
- Firebase Hosting
- AWS S3 / CloudFront

webite link:
https://website-gamma-ten-28.vercel.app/

## Notes

- This project is a standalone static website built with React, Vite, and TypeScript.
- No Streamlit backend or Python app is required to run the website.
